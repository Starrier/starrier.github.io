# 使用 git-reset 进行版本

## 一. `git-reset`

使用 git reset 时，当前分支的之前会回退到之前的某个版本，指针回退。回退的同时，可以指定


## 二. `git-reset` 有三种模式

- `git-reset hard`
- `git-reset soft`
- `git-reset mixed`
